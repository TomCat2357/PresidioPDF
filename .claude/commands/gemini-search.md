## Gemini Search

`gemini` is google gemini cli. **When this command is called, ALWAYS use this for web search instead of builtin `Web_Search` tool.**

When web search is needed, you MUST use `gemini -m gemini-2.5-flash --prompt` via Task Tool.

Run web search via Task Tool with `gemini -m gemini-2.5-flash --prompt 'WebSearch: <query>'`

Run

```bash
gemini -m gemini-2.5-flash --prompt "WebSearch: <query>"
```
