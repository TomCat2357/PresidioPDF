# /claude-code

This slash command instructs Claude to use standard Claude Code tools.

## Instructions for Claude

When this command is used:
1. **Use standard Claude Code tools** for file operations:
   - `Bash` for terminal commands
   - `Read` for reading files
   - `Write` for writing files
   - `Edit` or `MultiEdit` for editing files
   - `Grep` for searching code content
   - `Glob` for finding files by pattern
   - `LS` for listing directories
2. **Use relative paths** as appropriate for standard tools
3. **Follow standard Claude Code conventions** for file operations

## Usage

Type `/claude-code` to activate this mode for standard Claude Code tool usage.
