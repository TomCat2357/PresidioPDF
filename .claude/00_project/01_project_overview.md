# PresidioPDF プロジェクト概要

## プロジェクトの目的
日本語の個人情報検出・マスキングツールのPDF文書向け実装。Microsoft Presidioを活用したセキュリティとプライバシー保護のためのツール。

## コアバリュー
- **プライバシー保護**: 日本語個人情報の正確な検出と適切なマスキング
- **使いやすさ**: CLIとWebインターフェース両方の提供
- **高精度**: 複数のspaCy日本語モデル対応による柔軟な精度調整
- **拡張性**: モジュラー設計による機能拡張の容易さ

## 技術スタック
- **言語**: Python 3.8+
- **依存関係管理**: uv
- **NLP**: spaCy (日本語モデル), Microsoft Presidio
- **PDF処理**: PyMuPDF
- **Webフレームワーク**: Flask
- **テスト**: pytest
- **品質**: black (フォーマット), mypy (型チェック)
